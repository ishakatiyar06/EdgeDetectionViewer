Open this folder in Android Studio:
1. File -> Open -> select the 'android' folder in this project.
2. Let Gradle sync. Ensure Android SDK, NDK and CMake are installed (use SDK Manager).
3. Connect a physical device or use an emulator with camera support (physical device recommended).
4. Run the app.
5. Tap 'Process Sample' to load the packaged sample PNG, process with native code, and display the result.
